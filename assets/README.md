# Assets Directory

Este directorio debe contener los siguientes archivos para que la app funcione correctamente:

## Archivos Requeridos

1. **icon.png** (1024x1024px)
   - Icono de la aplicación para iOS y Android
   - Formato: PNG con fondo transparente o sólido

2. **splash.png** (1284x2778px recomendado)
   - Imagen de splash screen
   - Se mostrará al iniciar la app

3. **adaptive-icon.png** (1024x1024px)
   - Icono adaptativo para Android
   - Debe tener un área segura de 432x432px en el centro

4. **favicon.png** (48x48px)
   - Favicon para la versión web

5. **notification-icon.png** (96x96px, opcional)
   - Icono para las notificaciones push

6. **notification-sound.wav** (opcional)
   - Sonido personalizado para las notificaciones

## Generación de Assets

Puedes usar herramientas como:
- [Expo Asset Generator](https://www.npmjs.com/package/@expo/asset-generator)
- [App Icon Generator](https://www.appicon.co/)
- [Figma](https://www.figma.com/) para diseñar los iconos

## Nota

Si no tienes estos assets aún, la app funcionará pero necesitarás agregarlos antes de publicar en las tiendas.
